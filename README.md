# Supporter HTML5 Mobile App #

## Tech Stack ##

* Node.js
* Backbone
* Underscore
* Handlebars
* jQuery/Zepto
* Foundation
* Less/Sass

## Getting Started ##

```
npm install
npm start
```
Now open: http://localhost:3000
